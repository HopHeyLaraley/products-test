Данный проект представляет собой простую систему управления товарами. 
Он позволяет работать с коллекцией товаров: просматривать, изменять их количество, скрывать и настраивать лимит отображения.

Возможности:
-Просмотр списка всех товаров.
-Увеличение и уменьшение количества товаров.
-Скрытие товара из списка.
-Настройка лимита отображения товаров.

Файлы проекта:
-index.php - главная страница для отображения списка товаров.
-handler.php - обработчик запросов для изменения количества, скрытия товаров.
-db.php - настройки подключения к базе данных.
-CProducts.php - класс с методами для работы с таблицей товаров.
-create_table.sql - SQL-скрипт для создания таблицы products в базе данных.
-config.php - файл конфигурации, позволяющий менять параметры для работы системы (сейчас только лимит строк таблицы).

Использование SQL-скрипта:
Файл create_table.sql используется для создания таблицы products. Он должен быть выполнен в MySQL перед началом работы с проектом.

Как пользоваться:
Убедитесь, что выполнен SQL-скрипт для создания таблицы.
Настройте подключение к базе данных в файле db.php.
Перейдите на страницу index.php для работы с системой управления товарами.
